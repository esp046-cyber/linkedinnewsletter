# linkedinnewsletter
ChrisTechVA Newsletter Studio — single HTML file, production-ready PWA.
What's inside:
Generator — select 50–300 posts, choose tone & length, progress bar during generation
Library — full search, topic filter, status filter (saved/used/unused), paginated 20/page
Topics — 12 AI automation topic cards, click to browse filtered posts
Export — Copy All, TXT, JSON, CSV, Favorites-only
Settings — toggles for numbers/badges/compact view, clear/reset actions
Each newsletter covers: workflow automation, lead gen, AI agents, API integrations, CRM, marketing pipelines, ops, productivity, n8n, startups, strategy — all authored around ChrisTechVA as the authority voice with LinkedIn hashtags and CTAs.
PWA features: Service worker, installable manifest, offline-capable, mobile-first with amethyst→mint gradient design system.
